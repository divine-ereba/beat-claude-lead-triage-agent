# AI Automation Intern 012 – Lead Triage Agent

This repository contains my submission for the **Single Grain AI Automation Intern 012 Challenge (Brief Version: 2026-07)**.

The project implements an AI-powered Lead Triage Agent using **n8n** and **OpenAI** to automate the first-pass review of inbound website leads while maintaining human oversight for security, compliance, and operational risks.
